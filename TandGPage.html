T&G Holiday Promotion Page

Overview

While working as a marking intern at Taylor & Grace Marketing (T&G), I was assigned a task to design and launch a landing page on their website for a promotion they were running. The page had to include a unique URL, outline of what is included in the promotion, client testimonials, a countdown timer, and a sign up form. With these specififations, I brainstormed, implemented, and deployed the page while following a structured timeline to ensure timely updates and alignment with goals. Being receptive to feedback, the final result exceeded client approval and resulted in numerous new marketing opportunities for T&G!

As a result of my work on this project, T&G added web design as a new service—one that I will be leading by creating future web pages for clients.

Tools

Based on the assignment, I concluded on using the tools:

  • Wordpress
  • Elementor
  • Canva

My Role

Due to this assignment being individual, I assumed the following roles:

  • Website and UX/UI Designer creating wireframes and mackups
  • Front-End Developer with Wordpress and Elementor
  • Client Consultant by communicating and implementing client feedback

Brainstorming

Using the current homepage of T&G's website, I was able to get an understanding of the brand style and voice for the page I was going to design. I then utilized Canva and existing digital assets, such as brand colors, typography, and imagery, to create a wireframe that outlined the page structure. Once the layout was finalized, I developed a mockup that aligned with the brand's aesthetic.

To gather feedback, I presented the mockup to the client for approval, incorporating their initial input into refinements. Using virtual sticky notes, I documented key thoughts and suggestions along the sides of the design, making it easy to track revisions and ensure alignment with the T&G's vision. This collaborative process helped streamline decision-making and resulted in a final design that met both branding and functionality goals.

Implementation

After finalizing the wireframe and mockup, I implemented the design into a live webpage using WordPress, leveraging Elementor as the primary page builder. Since the T&G website was already hosted on WordPress, I worked within the existing infrastructure to ensure seamless integration.

To bring the design to life, I followed these steps:

  • Set up the page structure in Elementor according to the wireframe layout
  • Imported brand assets, including fonts, colors, and imagery
  • Customized sections and widgets to match the mockup
  • Incorporated interactive elements, such as buttons and hover effects, to enhance UX
  • Tested and adjusted the layout across various screen sizes to ensure proper responsiveness

Once the page was fully built, I conducted a final review with the client, making minor refinements as needed. This process resulted in a polished, professional webpage that aligned with T&G's branding and user experience goals.

Next Steps

While the project was successfully implemented, I wish I had more time to improve the content spacing and layout, especially towards the bottom of the page.

The placing of each element could have been cleaner and allowed for more intentional spacers, but I faced some challenges with uploading images in the correct font, which took vital time away from last touches before launching. If I were to revisit this project, I would focus on fine-tuning the spacing and alignment to enhance the overall polish of the design.

Despite these challenges, the page effectively captured the intended brand style and functionality.

Learnings

Overall, this experience taught me a lot about web design, working within existing brand guidelines, and using WordPress with Elementor. This was my first time fully implementing a design from wireframe to a live webpage for a client, and I gained valuable insight into translating static mockups into functional, interactive pages. I also learned the importance of problem-solving when facing technical challenges, such as image formatting and font inconsistencies. Beyond the technical skills, this project strengthened my ability to collaborate with clients and adapt to feedback. Moving forward, I will apply these lessons to future web design projects and continue refining my skills.
